Key Type Description
userId dbid_t %Database ID of the current user
userType qint16 Type of the current user as integer, see \link AdminAccount::AdminAccountType AdminAccountType \endlink
userName QString User name of the current user
userMaxDisplay quint8 Maximum display setting for lists for the current user
userWarnLevel quint8 Warn level in percent for the current user to display warnings on limits like quota
userTz QByteArray IANA time zone ID of the current user
template QString name/path of the template file
status_msg QString Status message for the last performed action. This key is only available, if a status message has been generated.
error_msg QString Error message if an error occured in the last performed action. This key is only availabe, if an error message has been generated.
